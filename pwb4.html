System Implementation Information

Aplikasi ini digunakan untuk mengelola implementasi dan pemeliharaan perangkat lunak.
Adapun kemampuan aplikasi ini sebagai berikut:

  1. Merekam aktivitas implementasi perangkat lunak
  2. Merekam data transaksi model baik fitur input maupun label output
  3. Merekam data lingkungan yang relevan dengan konteks implementasi perangkat lunak
  4. Merekam masukan.catatan dari pemelihara perangkat lunak
  5. Memantau kinerja perangkat lunak terpasang
